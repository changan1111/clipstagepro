║         ClipStage — Editor Setup (One Time)          ║
╚══════════════════════════════════════════════════════╝

SETUP (do this once on each editor Mac):
─────────────────────────────────────────
1. Copy "ClipStage.app" to your Desktop (or Applications folder)
2. Right-click → Open (first time only, to allow it)
3. From next time: just double-click normally

DAILY USE:
──────────
1. Double-click ClipStage.app
2. It connects, mounts your folders, opens the browser
3. Select your name → search → stage → drag to FCP

FCP WORKFLOW:
─────────────
1. Search and stage clips in ClipStage
2. In FCP: File → Import → Media
3. Navigate to /Volumes/staging/YourName/
4. Select clips → IMPORTANT: choose "Leave Files in Place"
5. Done — clips link directly from the archive, no copying

IF SOMETHING GOES WRONG:
─────────────────────────
• "Cannot reach server" → Check studio network / ask supervisor
• Clips show ⚠️  in staging → Archive volume may be offline
• Browser shows blank → Wait 30 seconds and refresh

DO NOT:
────────
• Do not move or rename ClipStage.app
• Do not close the browser tab while staging
• Do not quit the app from Dock — just close the window

Support: Inform your supervisor. Do not try to restart anything yourself.
